## Gnome Terminal

En **Ubuntu/Linux** usa **Gnome Terminal** punto.

***Nota:*** Lo mejor no tieness que instalar nada, ya viene por defecto en Ubuntu.


## Terminales aceleradas por GPU

### 1. Alacritty

***Pre-requisito:*** Instala rust y cargo [**aquí**](/languages/rust/).

```bash
cargo install alacritty
```

Para más información visita [**documentación de alacritty**](https://alacritty.org/).

### 2. Kitty

```bash
curl -L https://sw.kovidgoyal.net/kitty/installer.sh | sh /dev/stdin
```

Para más información visita [**documentación de kitty**](https://sw.kovidgoyal.net/kitty/).
