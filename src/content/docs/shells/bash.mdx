---
title: Bash
description: Bash es un lenguaje de programación de scripts para el shell de Unix. El nombre es un acrónimo de Bourne-Again SHell, una referencia al hecho de que Bash es una reemplazo de código abierto para el Bourne Shell original y el Korn Shell. Es compatible con la mayoría de las características del Korn Shell (KSH) y el C Shell (CSH), con algunas características de Bash original y el Shell TENEX. Es también el shell de usuario predeterminado en macOS.
