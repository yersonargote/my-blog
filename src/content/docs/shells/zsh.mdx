**ZSH** es un shell de Unix que se puede usar como intérprete de comandos de inicio de sesión interactivo y como lenguaje de scripting de shell. ZSH amplía las características de Bourne Shell (sh) y Korn Shell (ksh) con nuevas características útiles y un gran número de complementos.

## Instalación

Para instalar ZSH en Ubuntu, ejecute el siguiente comando:

```bash
sudo apt install zsh
```

## Instalar Zap

**Zap** es un gestor de plugins zsh mínimo que hace lo que esperas.

```bash
zsh <(curl -s https://raw.githubusercontent.com/zap-zsh/zap/master/install.zsh) --branch release-v1
```

**Nota:** Si desas más información sobre Zap, puedes visitar su [repositorio](https://github.com/zap-zsh/zap.git).
