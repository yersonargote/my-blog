## Windows Terminal

**Windows Terminal** es una moderna aplicación host para los shells de línea de comandos que ya te gustan, como Command Prompt, PowerShell y bash (a través de Windows Subsystem for Linux (WSL)).

### Instalación

***Recomendado***: Instalar desde la tienda oficial de Windows [**Microsoft Store**](https://aka.ms/terminal).


### Configuración

Si desea configurar **Windows Terminal** ver la documentación oficial [**aquí**](https://docs.microsoft.com/es-es/windows/terminal/).

## WSL 2

Eres programador y usas Windows 11, entonces debes usar Linux y **WSL 2** es la mejor opción.


### Instalación

Si tienes **Windows 11** solo ejecuta el siguiente comando en PowerShell como administrador:

```powershell
wsl --install
```

Si tienes problemas de instalación, puedes seguir la guía oficial de Microsoft [**aquí**](https://docs.microsoft.com/es-es/windows/wsl/install).
