## Windows Terminal

Usa **Windows Terminal** para tener una mejor experiencia en la terminal de Windows 11.

### Instalación

Instala **Windows Terminal** desde la tienda oficial [**Microsoft Store**](https://aka.ms/terminal).

**Configuración**: Si desea configurar o tiene algún problema ver la documentación oficial [**aquí**](https://docs.microsoft.com/es-es/windows/terminal/).

## WSL 2

Usa **WSL 2** para tener **Ubuntu** en Windows 11.

### Instalación

Si tienes **Windows 11** solo ejecuta el siguiente comando en PowerShell como administrador:

```shell
wsl --install
```

**Nota 1:** El comando anterior instalará **WSL 2** y **Ubuntu**.

**Nota 2:** Si tienes problemas de instalación, puedes seguir la guía oficial de Microsoft [**aquí**](https://docs.microsoft.com/es-es/windows/wsl/install).
