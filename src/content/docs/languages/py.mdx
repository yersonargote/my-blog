**Python** es un lenguaje de programación interpretado cuya filosofía hace hincapié en la legibilidad de su código. Se trata de un lenguaje de programación multiparadigma, ya que soporta orientación a objetos, programación imperativa y, en menor medida, programación funcional. Es un lenguaje interpretado, usa tipado dinámico y es multiplataforma.

## Instalación

**Nota:** Nos enfocaremos en la instalación en [**Linux**](/terminals/linux/), si usas Windows te recomiendo que uses [**WSL2**](/terminals/windows/).

```bash
sudo apt install python3
```

Verificamos la instalación:

```bash
python3 --version
```

### Instalar pip

```bash
sudo apt install python3-pip
```

Verificamos la instalación:

```bash
pip3 --version
```

### Instalar virtualenv

```bash
sudo apt install python3-virtualenv
```

### Instalar venv

```bash
sudo apt install python3-venv
```
