**Java** es un lenguaje de programación de propósito general, concurrente y orientado a objetos. Su intención es permitir que los desarrolladores de aplicaciones escriban el programa una vez y lo ejecuten en cualquier dispositivo (conocido en inglés como WORA, o "write once, run anywhere").

## Instalación

**Nota:** Nos enfocaremos en la instalación en [**Linux**](/terminals/linux/), si usas Windows te recomiendo que uses [**WSL2**](/terminals/windows/).

### Instalar SDKMAN!

La mejor forma de instalar java es usando [**SDKMAN!**](https://sdkman.io/), para instalarlo solo debes ejecutar el siguiente comando:

```bash
curl -s "https://get.sdkman.io" | bash
```

### Instalar java

Una vez instalado, debes cerrar y abrir la terminal nuevamente, luego ejecuta el siguiente comando para instalar java:

```bash
sdk install java
```

**Nota:** Verifica que se haya instalado correctamente ejecutando el siguiente comando:

```bash
java --version
```

### Más

Con **SDKMAN!** también puedes instalar otras versiones de java, para ver las versiones disponibles ejecuta el siguiente comando:

```bash
sdk list java
```

**Nota:** Sales de la lista con la tecla `q`.

Para instalar una versión específica ejecuta el siguiente comando:

```bash
# Recomiendo la ultima versión LTS con graalvm
sdk install java <version>-graalce
# Ejemplo:
sdk install java 21-graalce
```

O también puedes instalar **Gradle** o **Maven** para manejar tus proyectos:

```bash
sdk install gradle
# o
sdk install maven
```
