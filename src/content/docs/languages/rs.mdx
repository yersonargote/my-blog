**Rust** es un lenguaje de programación de sistemas que se centra en la seguridad, especialmente en la concurrencia, la velocidad y la estabilidad de la memoria. Rust es sintácticamente similar a C++, pero proporciona mayor seguridad de memoria sin tener un recolector de basura.

## Instalación

**Nota:** Nos enfocaremos en la instalación en [**Linux**](/terminals/linux/), si usas Windows te recomiendo que uses [**WSL2**](/terminals/windows/).

```bash
curl --proto '=https' --tlsv1.2 -sSf https://sh.rustup.rs | sh
```

Para verificar que se instaló correctamente, ejecuta el siguiente comando:

```bash
rustc --version
```

**Nota:** Si desea más información sobre la instalación, puede consultar la [documentación oficial](https://www.rust-lang.org/tools/install).
